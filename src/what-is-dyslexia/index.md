# ディスレクシアとは
ディスレクシアは、知的能力には異常がないにもかかわらず、文字の読みもしくは書き（もしくはその両方）に著しい困難を示す障害のことです。この困難さは「音韻情報処理過程」や「視覚情報処理過程」などの障害によるものと考えられており、人によってその困難さの状態は[さまざまです。](./cases.md)
1999年に文部科学省が定義した「学習障害」の中核をなす障害であると考えられます。
